EVE Data Interface

Introduction

EVE Data Interface is a C++ Interface to read data written with eveCSS It consists of an API description, which must be included into the the users C++ application and a shared library, which will be loaded at run-time.

EVEH5 shared library

The EVEH5 shared library supports HDF5 data files and needs a HDF5 library to compile. Since it uses features from C++11, it must be compiled with -std=c++11 (for build command see eveH5.pro).

Usage

  • Include eve.h in your application and link against the shared library
  • EVEH5 supports EVEH5 layout from version 1 to version 5
